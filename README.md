# IMiP_openUC2_Motor_Rotation_Scan
Python-based motor control for automated sample rotation in 3D refractive index tomography. Uses openUC2 hardware and XIAO ESP32-S3 via JSON serial commands. Part of an off-axis Mach-Zehnder holographic imaging pipeline built for Innovation Methods in Photonics 2026, University of Jena, Germany.
